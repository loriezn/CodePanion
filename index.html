{ CodePanion }

github for web developers



    


    

      
{ What is this? }

      
CodePanion is an online tool that allows you to commit web files to github. It allows the use of online editors, such as CodePen, to commit files to a repository on GitHub. Now you
        can code and commit anywhere!


      
{ What can I commit? }

      
You can commit any file or resource that has a web filepath. However, all URL's are subject to Cross-origin resource sharing restrictions.
